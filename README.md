# Hotel Aqua

Website for Hotel Aqua, a hotel in Kiten, Bulgaria.

## Tech Stack

Next.js 15 · React 19 · Tailwind CSS v4 · react-slick

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Pages

- **/** — Home page
- **/about** — About the hotel
- **/our_hotel** — Hotel details
- **/prices** — Pricing
- **/gallery** — Photo gallery with category filtering
- **/contacts** — Contact information
- **/booking** — Booking form
